# Liventory — Vision Document

Static vision page for [Liventory](https://liventory.vercel.app), an AI agent operating system for nursery and tree-growing businesses.

## What this is

A single-page vision document outlining the Liventory platform — six autonomous AI agents designed to handle crop intelligence, carbon tracking, grant discovery, business operations, and field work for independent nurseries.

Built as a standalone HTML file and deployed via Vercel.

## Stack

- Plain HTML/CSS/JS — no framework, no build step
- Deployed on Vercel
- Fonts via Google Fonts (Playfair Display, Syne, Crimson Pro)
